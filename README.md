# 🔍 Smart Wiki Search

A dynamic and intelligent Wiki Search application designed to deliver fast, relevant, and user-friendly search results using real-time API integration.

## 🚀 Live Demo
👉 https://sksearch.ccbp.tech/

---

## ✨ Features

- 🔎 **Smart Search**
  - Real-time keyword-based search
  - Instant results with API integration

- ⚡ **Fast Performance**
  - Optimized data fetching and rendering
  - Smooth user experience

- 🎯 **User-Friendly Interface**
  - Clean and minimal UI
  - Fully responsive (mobile + desktop)

- 🌐 **Seamless Navigation**
  - Clickable result titles and URLs
  - Easy exploration of topics

- ⏳ **Loading Indicator**
  - Spinner for better user feedback during API calls

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **API:** Wiki Search API  
- **Concepts Used:** DOM Manipulation, Fetch API, Event Handling

## ⚙️ How It Works

1. User enters a keyword and presses **Enter**
2. Application sends a request to the API
3. Displays results dynamically:
   - Title (clickable)
   - URL
   - Description
4. Shows a loading spinner while fetching data
